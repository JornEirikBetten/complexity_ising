# Complexity investigation in the Ising Model 
Investigation of complexity in Ising model phase transitions. 



## Usage

Note that you need to install `jaxlib` in addition to the packages written in `requirements.txt` according to your execution environment.

```sh
$ pip install -U pip && pip install -r requirements.txt
$ pip install --upgrade "jax[cuda11_pip]" -f https://storage.googleapis.com/jax-releases/jax_cuda_releases.html
```
